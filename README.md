# minato-makoto.github.io

This repository hosts the source for Minato Makoto's personal website, served through GitHub Pages. It consists of static HTML/CSS files.

## Setup and Usage

1. Clone this repository.
2. Open `index.html` directly in your browser **or** start a simple local server and navigate to `http://localhost:8000`:
   
   ```bash
   python3 -m http.server 8000
   ```
